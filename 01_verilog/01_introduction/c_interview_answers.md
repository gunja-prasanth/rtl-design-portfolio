# Interview Answers

## Q1. What is Verilog?

Verilog is a Hardware Description Language (HDL) used to model, design, simulate, and verify digital circuits.

---

## Q2. Why is Verilog used?

Verilog is used to describe hardware behavior and structure before manufacturing. It helps engineers simulate and verify designs to reduce hardware errors.

---

## Q3. Is Verilog a programming language?

No. Verilog is a Hardware Description Language. It describes hardware that operates concurrently, unlike software programming languages that execute sequentially.

---

## Q4. What is the difference between Verilog and VHDL?

Verilog has a C-like syntax and is generally easier to learn. VHDL is more verbose and strongly typed. Both are used for digital design.

---

## Q5. What is RTL in Verilog?

RTL (Register Transfer Level) is a design abstraction where data movement between registers and the logic operating on that data are described.

---

## Q6. What are the major applications of Verilog?

* FPGA Design
* ASIC Design
* Processor Design
* Communication Protocol Design
* Memory Design
* Verification

---

## Q7. What is the typical Verilog design flow?

Specification → RTL Coding → Simulation → Synthesis → Implementation → Hardware Testing

---

## Q8. What is the difference between Verilog and SystemVerilog?

SystemVerilog is an extension of Verilog. It adds new data types, interfaces, assertions, classes, and verification features.

---

## Q9. Why do companies use simulation before synthesis?

Simulation helps detect design bugs before hardware implementation, reducing development cost and debugging effort.
