# RTL Design Portfolio

## Overview

This repository documents my journey toward becoming a VLSI Frontend Engineer. It contains Verilog RTL designs, SystemVerilog concepts, protocol implementations, verification projects, testbenches, and interview preparation notes developed during my placement preparation.

The goal of this repository is to build strong fundamentals in Digital Electronics, RTL Design, Verilog, SystemVerilog, Protocols, and Verification while maintaining a consistent engineering portfolio.
